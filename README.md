<h2>Installation</h2>

<p>You need install Go 1.20 or higher version, You can download from <a href="https://golang.org/dl/">here</a>.</p>

<h2>Usage</h2>

```bash
make run
```
<br />

You pass custom config file via define "CONFIG_FN" environment variable
```bash
export CONFIG_FN="custom.json"
make run
```
