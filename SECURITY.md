# Security Policy

## Supported Versions

The following versions of Inzibat are currently supported with security updates:

| Version | Supported |
| ------- | --------- |
| v0.3.x  | ✅ Yes    |
| < v0.3  | ❌ No     |

We recommend always updating to the latest stable release of Inzibat.

## Reporting a Vulnerability

If you discover a security vulnerability in Inzibat, please **do not** report it using the public issue tracker. Instead, report it privately to the maintainer:

* **Email**: [me@lynicis.dev](mailto:me@lynicis.dev)

Please include the following details in your report:

* A description of the vulnerability and its potential impact.
* Detailed steps or a proof-of-concept (PoC) script to reproduce the issue.
* The version of Inzibat and environment details (OS, architecture) where the vulnerability was observed.

### Our Commitment

Upon receiving your report:

1. We will acknowledge receipt of your vulnerability report within **48 hours**.
2. We will work to verify the vulnerability and determine the impact.
3. We will coordinate a fix and release a patch version as soon as possible.
4. We will credit you for the discovery in the release notes, unless you request to remain anonymous.
